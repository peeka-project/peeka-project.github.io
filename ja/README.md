# Peeka 日本語ドキュメント

このディレクトリには Peeka ドキュメントの日本語版が含まれています。

## 構造

```
ja/
├── index.md             # 日本語トップページ
├── commands/            # コマンドリファレンスページ
│   └── index.md         # コマンド概要
└── README.md            # このファイル
```

## 翻訳ステータス

現在利用可能:
- ✅ ホームページ (index.md)
- ✅ コマンド概要 (commands/index.md)
- ✅ インストールガイド (installation.md)
- ✅ クイックスタートガイド (quickstart.md)
- ✅ アーキテクチャドキュメント (architecture.md)
- ✅ TUI ガイド (tui.md)
- ✅ トラブルシューティング (troubleshooting.md)
- ✅ すべてのコマンド個別ドキュメント

## 言語切り替え

ユーザーはナビゲーションバーの言語セレクターから言語を切り替えることができます:
- **中文** - 中国語ドキュメント（ルート）へのリンク
- **English** - 英語ドキュメント `/en/` へのリンク
