## Cambio de idioma

Los usuarios pueden cambiar el idioma desde el selector de idioma en la barra de navegación:
- **中文** - Enlace a documentación en chino (raíz)
- **English** - Enlace a documentación en inglés `/en/`
- **日本語** - Enlace a documentación en japonés `/ja/`
