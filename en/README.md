## Contributing Translations

We welcome contributions to translate the documentation! If you'd like to help:

1. Fork the repository
2. Create a new branch for your translation
3. Translate the markdown files from the Chinese version
4. Submit a pull request

Please maintain the same frontmatter structure and file organization as the Chinese version.

## Language Switcher

Users can switch between languages using the language selector in the top navigation:
- **中文** - Links to the Chinese documentation (root)
- **English** - Links to `/en/` for English documentation
